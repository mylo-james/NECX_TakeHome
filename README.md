## EZ-Start with Docker

1. Clone the repo,
2. cd into NECX_TakeHome
3. ```bash
   docker compose up
   ```
4. Wait about a minute

- MongoDB will be seeded
- Frontend will be served from :3000
- Backend will be served from :5050

5. Have fun!
